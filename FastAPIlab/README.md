# FaspAPI сервер 

## Файловая систма

- FastAPI - папка с проектом
    - app - содержит сервер и моудли длля работы программы
        - main.py
        - model_unit.py
        - modev.env - для содержания разных версий модели
        - module_download.py
    - model
        - версия модели
        - model_onfo.json - содержит коэффициэенты и r2_score

## Создание виртуального окружения 

Для создание виртуального окружения [смотреть](https://github.com/ComebaLL/LLM)

## Запуск приложения

```bash
uvicorn app.main:app --reload
```

## Роботоспособность

Для проверки работы серверка мы можем написать с адресной строке

```
http//127.0.0.1:8000/predict?x1=1x2=2x3=3x4=4
```

или

```
http//127.0.0.1:8000/metrics
```

Чтобы посмотреть документация по серверу нужно ввести

```
http//127.0.0.1:8000/docs
```


для докера
